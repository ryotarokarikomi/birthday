# stop_watch
***ロボットシステム学(課題2)***  
ROS 2のサンプルコードです。  
[stop_watch_msgs]()でこのパッケージで使用する型を設定しています。  
ストップウォッチです。


## 使い方
* ターミナル1  
  以下のコマンドでtalkerを実行します。
  ```
  $ ros2 run mypkg talker
  ```

* ターミナル2  
  以下のコマンドでlistenerを実行します。
  ```
  $ ros2 run mypkg listener
  ```
